**Visit Website:** https://unsplash.com

**Search For:**
- "luxury watch"
- "premium timepiece"
- "rolex"
- "omega watch"

**Download:**
1. Click on any watch image
2. Click "Download" button
3. Select highest resolution

## STEP 2️⃣: Compress Images (5 minutes)

**Why?** Images need to be small and fast-loading.

**Visit:** https://tinypng.com

**Process:**
1. Go to TinyPNG website
2. Drag your downloaded images
3. It compresses them automatically
4. Click "Download" for each
5. Save compressed versions

**Result:** Images are 50-80% smaller! ⚡

## 📸 Image Names Matter

**Hero Carousel expects these exact names:**
```
/public/images/banners/
├── hero-1.webp  ← Image 1
├── hero-2.webp  ← Image 2
└── hero-3.webp  ← Image 3
```

**Different names?** Edit this file:
- `components/layout/hero-carousel.tsx`
- Change line 28: `backgroundImage: "/images/banners/hero-1.webp"`

## ❓ Common Issues

**Q: Images not showing?**
A: Check:
1. Folder is named exactly: `banners`
2. Images are named: `hero-1.webp`, `hero-2.webp`, `hero-3.webp`
3. Folder is in `/public/images/` not elsewhere
4. Run `npm run dev` again to refresh

**Q: Do I need .webp format?**
A: Preferred but .jpg or .png work too. Just update file names in code.

**Q: Can I use different names?**
A: Yes, but edit `components/layout/hero-carousel.tsx` and update the paths.

**Q: How big should images be?**
A: 1920x800px ideal. Between 100KB-500KB after compression.
