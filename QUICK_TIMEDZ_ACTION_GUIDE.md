## ❓ Common Questions

**Q: Where should I save images?**
A: `public/images/banners/` for hero images

**Q: What image format is best?**
A: WebP format (best quality/size ratio). JPG also works.

**Q: What size should images be?**
A: 1920x800px for banners, 800x800px for products

**Q: Can I use images from Google?**
A: Not recommended. Use Unsplash/Pexels/Pixabay (legally free)

**Q: How many images do I need?**
A: Minimum 3 for hero slider, but 5-7 is better
